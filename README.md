# Om Prakash Sahoo - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean design with smooth animations and hover effects
- **Smooth Navigation**: Sticky navbar with smooth scrolling between sections
- **Interactive Elements**: Hover effects, typing animation, and parallax scrolling
- **Professional Sections**:
  - Home with introduction and call-to-action
  - About with education details
  - Skills showcase
  - Projects portfolio
  - Certifications
  - Achievements
  - Contact information

## Tech Stack

- **HTML5**: Semantic structure
- **CSS3**: Modern styling with Flexbox and Grid
- **JavaScript**: Interactive functionality and animations
- **Font Awesome**: Icons
- **Google Fonts**: Poppins font family

## Color Palette

## Getting Started

1. **Clone or Download** the files to your local machine
2. **Open** `index.html` in your web browser
3. **Customize** the content as needed:
   - Update personal information in HTML
   - Modify colors in CSS variables
   - Add your actual resume file
   - Update social media links

## File Structure

```
portfolio/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Customization

### Personal Information
- Update name, title, and tagline in the Home section
- Modify education details in the About section
- Add/remove skills in the Skills section
- Update project descriptions
- Add your certifications and achievements
- Update contact information and social links

### Styling
- Change colors by modifying CSS custom properties
- Adjust animations by modifying transition durations
- Update fonts by changing the Google Fonts import

### Content
- Replace the placeholder image with your professional photo
- Add actual resume file and update download link
- Update social media links with your profiles

The website can be deployed to any static hosting service:
